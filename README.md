<h2 id="tasks">Задачи проекта</h2>

_**Общее описание:**_
<p>Тестирование программы, которая помогает заказать бургер в Stellar Burgers. Покрыть программу юнит-тестами. </p>

_**Покрыть тестами классы:**_
1. Bun
2. Burger
3. Ingredient
4. IngredientType
